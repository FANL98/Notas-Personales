```python title:zipFor.py
a = [1, 2]
b = ["Uno", "Dos"]
c = zip(a, b)
for numero, texto in c:
	print("Número", numero, "Letra", texto) #Número 1 Letra Uno #Número 2 Letra Dos
```

Al combinar `zip` con un ciclo `for` podemos iterar sobre dos listas en paralelo 🚶‍♂️🚶‍♀️.
