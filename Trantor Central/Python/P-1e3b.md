```python title:whileLista.py
x = ["Uno", "Dos", "Tres"]
while x:
	x.pop(0)
	print(x) #["Dos", "Tres"] #["Tres"] #[]
```

`while` puede tomar como condición un dato de tipo lista con el que va a iterar mientras la lista contenga valores.
