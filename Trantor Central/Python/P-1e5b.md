```python title:breakWhile.py
x = 6
while True:
	x -= 1
	print(x) #5 #4 #3 #2 #1
	if x == 1:
		print("¡Feliz año nuevo!") #¡Feliz año nuevo!
		break
```

Para ciclos `while` `break` es funcional cuando el código se ejecuta indefinidamente, este solo se detendrá cuando la condición que se cumpla 🐢.
