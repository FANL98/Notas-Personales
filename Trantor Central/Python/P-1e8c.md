```python title:LCFunciones.py
def cuadrado(x):
	return x ** 2
cuadradosLista = [cuadrado(x) for x in range(10)]
print(cuadradosLista) #[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
```

"List Comprehension" puede ejecutar una función en lugar de hacer una expresión.
