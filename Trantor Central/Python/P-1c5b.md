```python title:concatenacionDeString
nombre = 'Francisco'
apellido = 'Angel'
nombre_completo = nombre + ' ' + apellido
print(nombre_completo)  # Salida: Francisco Angel
```

La concatenación se refiere a la union de "strings" en uno solo mediante el uso del singo `+`.
